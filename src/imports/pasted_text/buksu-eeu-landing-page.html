Design a bold, premium, and fully responsive landing page for BUKSU EEU — an official student ordering app for Bukidnon State University. Students use the app to browse and order school merchandise such as PE uniforms (top and bottom), ID lanyards, and other school items. Once the admin approves and prepares the order, students pick it up in person at the school office on campus. Color Palette — use with depth, never flat: #010837 — darkest navy, primary background base #100E3F — secondary background for alternating sections #0515AB — primary accent for buttons, highlights, icon backgrounds #8990CB — secondary accent for glows, gradients, subtle UI elements #DBDDE7 — body text, light surface elements #4F6EF7 — electric blue for gradient accents, card borders, CTA glow effects Use radial glows, layered gradients, and subtle glowing borders throughout to add visual depth. Avoid flat, monotone sections — every section should feel distinct but cohesive. Icon Rules — strictly consistent: Use only Lucide Icons across the entire page — line style, same stroke weight (1.5px), same size (24px). No emoji icons, no filled icons, no mixing of styles. Suggested icons: ShoppingCart (Easy Ordering), ClipboardCheck (Admin Approval), MapPin (Campus Pickup), Search (Browse), Send (Place Order), Clock (Wait), CheckCircle (Pick Up). Typography: Use Plus Jakarta Sans or Inter. Headlines: 700–800 weight. Body text: 400 weight in #DBDDE7. Maintain a consistent type scale across all sections. Minimum 16px for body text. Layout: Mobile-First + Responsive Design mobile-first at 375px base width. Every section must reflow into a single column on mobile with no horizontal scrolling. Buttons must have a minimum 48px tap height and be full-width on mobile. Also provide the desktop layout at 1440px. Section-by-Section: 1. Navbar Logo on the left, app name "BUKSU EEU" beside it.
